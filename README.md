# SVG to PNG

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

This repository is part of the **[Makr.io](https://makr.io)** collection—a set of 15 Next.js projects I created with the help of AI tools, primarily **Claude.ai** and **ChatGPT**. Each project was built to be adaptable, allowing anyone to learn from or modify the code for their own use.

## Overview

**SVG to PNG** is a Next.js app designed to quickly and efficiently convert SVG files to PNG format. It’s lightweight, responsive, and developed with a mix of coding and AI-assisted code generation, making it a useful tool for designers, developers, and anyone working with SVG images.

## Features

- **Convert SVG to PNG** – Easily upload an SVG file and convert it to a PNG format with just one click.
- **Fully Responsive** – Designed for smooth performance across devices, from desktops to mobiles.

## Getting Started

Clone this repository and install the dependencies:

```bash
git clone https://github.com/renedeanda/svg2png.makr.io.git
cd svg2png.makr.io
npm install
```

### Running Locally

To run the project locally:

```bash
npm run dev
```

Access the app at `http://localhost:3000`.

### Building for Production

To build for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
